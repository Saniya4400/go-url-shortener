\# 🚀 Go URL Shortener



A fast and simple URL Shortener built using \*\*Go (Golang)\*\*.  

It converts long URLs into short links and redirects users to original URLs.



\---



\## ✨ Features



\- 🔗 Shorten long URLs

\- 🔄 Redirect short URLs

\- 💾 JSON file storage (persistent data)

\- ⚡ Fast Go HTTP server

\- 🧠 Random short code generator



\---



\## 🛠️ Tech Stack



\- Go (Golang)

\- net/http package

\- JSON file storage
